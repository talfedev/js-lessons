Tiny's lessons so far

Lesson 1

  • Keywords:
    Keywords are reserved words that are use by the language and cannot be used as the names for variables and functions, some keywords that are often used are:
    - let
    - const
    - if
    - else
    - true
    - false
    - for
    - while
    - var
    - this
    - in

    and there are more we'll learn about as we progress

  • variables:
    const - for constant values, should be written in ALL_CAPS
    let - for changing values, should be written in camelCase
    naming - letters, numbers and underscore. cannot start with a number or contain a '-'.
    cannot redeclare variables

  • Conditions:
    if (conditon) {
    // excute code block 1
    } else if(condition) {
    // execute code block 2
    } else {
    // execute code block 3
    }

  • data types:
    number
    string
    boolean
    undefined
    null
    bigint
    object
    symbol

  • Logical operators:
    && - AND, only true if both sides of it are true
    || - OR, only false if both sides of it are false
